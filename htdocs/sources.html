Sources
This website could not be possible without these helpful and useful resources.

Tdecker91's Visualcube

Tdecker91's typescript fork of Visualcube is a really useful program that can create many different types of rubik's cube images. This was an essential program to almost all the images in the website. The ability to pass in an algorithm and add arrows helped with visuallizing the moves and was integral to the notation page.

Jperm

Jperm's youtube channel is what inspired me to start cubing. His tutorials are really easy to follow and well made and I highly encourge you to check them out. His website jperm.net also was the isparation for this website. Almost all the algorithms I learned was from his website. Without it, this website wouldnt have existed.
